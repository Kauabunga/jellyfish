## Prerequisites

- [Git](https://git-scm.com/)
- [Node.js](nodejs.org) Node >=4.X.X
- [Java](java.org) JDK >=7

## Get going

1. `git clone https://github.com/solnet-aquarium/jellyfish.git`
2. `cd jellyfish`
2. `npm install`
3. `npm run setup`
4. `npm test`
5. `npm start`


## Create a word document

`npm run create-document`
